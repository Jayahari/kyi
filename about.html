Know Your India is a submission for the OpenData Challenge from data.gov.in. This was developed as a teaching aid for kids so that they can visualize what they learn!!

Developed by Aravindhan Raju and Jayahari Jayaraman - Contact jayahari001[a@t]gmail[d.o.t]com

Map files are generated from Open Street Maps ( SHP files from http://download.geofabrik.de/ ), parts from Maptell.com and two maps from WikiPedia.

Javascript libraries used are from Kartograph.org, Bootstrap 2.3.2 , Datatables.net , Qtip2 , jQuery , Chroma.js and raphael.js ( Yup that's lot of libraries to make a site heavy )

If you wish us to add any dataset please mail to jayahari001[a@t]gmail[d.o.t]com with the dataset url from data.gov.in

If you like to use this project or report any bugs - visit out github project page

Developed on Firefox, Should work on Chrome, IE 7+ and Safari
